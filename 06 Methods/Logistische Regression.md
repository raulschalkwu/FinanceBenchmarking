# Logistische Regression (Logit)
**Intuition:** Lineares Modell für Wahrscheinlichkeiten; der Klassiker der [[Bankruptcy Prediction]] (Ohlson 1980).
**Stärken:** interpretierbar, stabil, regulatorisch akzeptiert. **Schwächen:** linear, Interaktionen manuell.
**Rolle:** Untere Messlatte in jedem Horse-Race ([[Predicting Financial Distress Using Machine Learning Techniques]], [[Earnings Forecast Accuracy and Implied Cost of Capital]]).
