# Interpretable Machine Learning
**Intuition:** Entweder inhärent interpretierbare Modelle oder Post-hoc-Erklärungen (SHAP, Feature Importance) für Black Boxes.
**Warum zentral für die WU:** Prüfung & Regulierung verlangen Nachvollziehbarkeit — Interpretierbarkeit ist hier kein Nice-to-have, sondern Publikationsstrategie ([[Explainable AI]]).
