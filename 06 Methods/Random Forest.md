# Random Forest
**Intuition:** Viele dekorrelierte Entscheidungsbäume auf Bootstrap-Stichproben, Mehrheitsvotum/Mittelung.
**Stärken:** robust, wenig Tuning, nichtlinear, Feature-Importances. **Schwächen:** keine Extrapolation, weniger präzise als Boosting, Black-Box-Tendenz.
**Finance:** Renditeprognose ([[Empirical Asset Pricing via Machine Learning]]), Distress ([[Predicting Financial Distress Using Machine Learning Techniques]]).
**Valuation:** Baseline für Gewinnprognosen ([[Fundamental Analysis via Machine Learning]]).
