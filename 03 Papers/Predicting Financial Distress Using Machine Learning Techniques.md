# Predicting Financial Distress Using Machine Learning Techniques

## Metadata
Autoren: Sethi, Singh, Gupta · Jahr: 2026 · Journal: Asia-Pacific Financial Markets (peer-reviewed) · Relevanz: Hoch · Link: https://link.springer.com/article/10.1007/s10690-025-09525-7

## Research Question
Prognostizieren ML-Modelle Financial Distress besser als klassische Verfahren?

## Data
1.111 NCLT-Unternehmen (Indien) → [[NCLT-Unternehmensdaten]].

## Methodology
[[Random Forest]], ANN ([[Deep Neural Network]]), [[Logistische Regression]] im Vergleich.

## Main Findings
ML schlägt klassische Verfahren; **Fehlklassifikation wird als Audit-Failure gerahmt** – der Auditing-Winkel.
