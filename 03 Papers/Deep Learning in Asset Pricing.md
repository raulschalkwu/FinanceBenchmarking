# Deep Learning in Asset Pricing

## Metadata
Autoren: Chen, Pelger, Zhu · Jahr: 2023 · Institution: [[Stanford GSB]] · Relevanz: Hoch · Link: https://economics.yale.edu/sites/default/files/deep_learning_in_asset_pricing.pdf

## Research Question
Wie modelliert ML den stochastischen Diskontfaktor?

## Data
US-Aktien + Makro-Zustandsvariablen.

## Methodology
Deep NN mit **No-Arbitrage-Bedingung** als ökonomischer Restriktion; adversariales Setup ([[GAN]]-Idee), [[Representation Learning]] für Zustandsvariablen.

## Main Findings
Ökonomische Struktur im Netz verbessert SDF-Schätzung; [[Stochastischer Diskontfaktor]] = Diskontfaktor unter ökonomischen Zuständen (Makro-Winkel zum Diskontsatz).

## Contributions
Blaupause für „Theorie + ML" statt reinem Data Mining → direkt relevant für [[Teaching Economics to the Machines]].
